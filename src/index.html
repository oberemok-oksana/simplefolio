Projects

"Weather forecast for your trips"

With the help of this application you can check the weather for your upcoming trip in 15 days. You can add a new trip from predefined list, chose your trip from a list or search it and see the weather forecast for the period of a trip and today's weather at that city. I created the app using React, Redux Toolkit, weather API. Also I used portal for modals.

"Meal-choice"

It’s a web app that suggests random meal ideas and recipes based on user preferences such as meal type, dietary restrictions, and cuisine. Users can browse through a recipe database and search for specific keywords or ingredients. It's a user-friendly tool for exploring new meal ideas and recipes while considering dietary restrictions and preferences. It was built in REACT, using CSS, TanStack Query library, collection of react-loader-spinners and translation module i18n.

See Live Source Code

"Subtitle-extractor"

It is my first useful project which I created to make it easy to read subtitles for learning English.It's a web app that extracts unnecessary time stamps, spaces and transforms text to lowercase. Users can upload a text and then copy it to the clipboard. It was built in Vanilla JS and CSS.
